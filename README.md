# Text as Data

Website for the **Text as Data** conference at the University of California, Berkeley (October 5, 2026).

## Pages

- **Home** — conference overview and key information
- **Call for Presentations** — submission guidelines and topics
- **Logistics** — venue, travel, registration, and contact

## Local preview

Open `index.html` in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## GitHub Pages

This site is set up for GitHub Pages from the `main` branch.

1. Push the repository to GitHub.
2. Go to **Settings → Pages** in the repo.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Choose branch **main** and folder **/ (root)**.
5. Save. The site will be available at:

   **https://denistillation.github.io/tada_website/**

Changes may take a minute or two to appear after pushing.
